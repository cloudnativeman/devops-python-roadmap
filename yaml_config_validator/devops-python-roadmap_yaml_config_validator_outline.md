# Project 3: YAML Config Validator

## Goal
Validate and update required fields in a YAML configuration file.

## Steps
1. Load YAML file.
2. Check for required fields (e.g., "version").
3. Prompt user to add missing fields.
4. Save updated YAML.

## Learning Points
- Working with YAML (`PyYAML`)
- File read/write
- Conditional logic
